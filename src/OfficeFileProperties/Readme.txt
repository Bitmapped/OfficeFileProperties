To test Microsoft Access databases, an MSI-based version of the Microsoft Access Database Engine must be installed.

Supported versions including a MSI-based installation of the full Microsoft Office or
an MSI-based redistributable such as the Access 2016 (https://www.microsoft.com/en-us/download/Confirmation.aspx?ID=13255)
or Access 2010 (https://www.microsoft.com/en-us/download/details.aspx?id=54920) versions.

Click2Run-based installs of Microsoft Office are not supported. The Office 365 Access Runtime installer is Click2Run-based and is not supported.
